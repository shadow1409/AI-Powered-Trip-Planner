# AI-Powered-Trip-Planner
A modular AI-assisted travel planning system that combines deterministic ML-based relevance scoring with controlled Gemini-powered language generation, orchestrated through a reproducible data pipeline and Streamlit interface. Designed with clear separation of logic, data, and presentation.
